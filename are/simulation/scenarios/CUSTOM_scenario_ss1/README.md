# Single-turn, single-step task 1
This scenario evaluates basic tool calling performance for an agentic system across a single turn with a single step. It serves more as a raw functionality test to ensure that the context cortex and tool calls are all functioning correctly. It targets a **single read** within an ARE-integrated dtabase app.

## Usage
### Basic Usage
To run the app tutorial scenario:
```bash
python -m are.simulation.scenarios.CUSTOM_scenario_ss1.scenario
```
